## 🚀 Getting Started

### Prerequisites
- GCC compiler installed
- Linux / macOS / WSL recommended

### Compile & Run a Solution

```bash
# Clone the repository
git clone https://github.com/Roshan-Mallick/leet-code-problems.git
cd leet-code-problems

# Compile any solution
gcc 01_easy/arrays/0001_two_sum.c -o two_sum

# Run it
./two_sum
```

## 📌 File Naming Convention

Each file follows this format:

```
{problem_number}_{problem_name}.c
```

**Example:**
```
0001_two_sum.c
0053_maximum_subarray.c
0300_longest_increasing_subsequence.c
```

> 💬 Every solution file contains comments explaining the approach, logic, and key steps.
